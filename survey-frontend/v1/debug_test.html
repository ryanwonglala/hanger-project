🐛 Common Issues & Solutions

  • Images not loading: Check that images folder contains hanger1.jpg, hanger2.png, hanger3.jpg
  • Carousel not rotating: Check browser console for JavaScript errors
  • Blackboard not appearing: Verify modal HTML is present in index.html
  • Theme not changing: Check CSS classes are applied to body element
